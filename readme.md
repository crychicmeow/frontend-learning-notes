# # Frontend Learning Notes 一个仿菜鸟教程风格的前端学习知识库

## 🚀 项目介绍

它面向零基础读者，希望它成为您在学习前端时的助手，致力于在2个月内找到一家100人以上的中厂前端实习岗位。

如果它能让您感到一丝「 原来前端也没有那么难」或者在您入职前提供了一点点帮助，我都会倍感荣幸。

同时这也是一个记录我学习前端基础的博客项目。从零开始，一步一个脚印，这里存放着我学习路上的所有笔记、代码和成长！我会把这里打造成最棒的知识小花园！

后续打算增加JS、Vue、Node甚至八股文等内容，全程不收费，完全开源免费。

## ✨ 目前已实现的功能

- **响应式导航栏**：一个漂亮的侧边栏，包含了所有学习章节，点击可以平滑滚动到对应位置。
- **精美的文章排版**：精心调整了字体、颜色、间距，让阅读体验更上一层楼。
- **代码语法高亮**：使用 `highlight.js` 对代码块进行语法高亮，代码从此五彩斑斓，清晰易读！
- **丰富的UI细节**：包括卡片式阴影、悬浮动效等，让页面充满设计感。

## 🛠️ 使用的技术

- **HTML5**：负责搭建页面的骨架。
- **CSS3**：负责美化页面的样式和布局。
- **JavaScript**：负责实现导航栏的交互和代码高亮功能。
- **Highlight.js**：一个超棒的第三方库，用来实现代码高亮。
